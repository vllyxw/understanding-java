- HEAD

- TRACE

SINTAXE:

- GET:
protected void doGet(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException
	{
		String nome = req.getParameter("nome");
		resp.getWriter().append("Bem vindo, voce acessou via POST");
	}
