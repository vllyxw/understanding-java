<h2>ARRAYS</h2>

<b>Arrays</b> = é uma lista de elementos, ordenado por indies começando por 0 como primeira posição

Exemplo 1: 

~~~
const lista = [10, "oi", true, [1,2], {a:b}];
~~~ 

Exemplo 2:

~~~
const notas = [10, 6.5, 8, 7.5];
consta media = (notas[0] + notas[1] + notas[2] + notas[3]) / notas.length;
console.log(media);
~~~

<b>Adicionando elementos com Arrays</b>

Exemplo: 

 ~~~
const notas = [10, 6, 8];

notas.push(7); //metodo push para adicionar elemento na array, adicionando o valor no final da array

consta media = (notas[0] + notas[1] + notas[2] + notas[3]) / notas.length;
console.log(media);
~~~

JavaScript permite atribuir um array dentro de uma const podendo alterar elas


<b>Excluindo elementos com Arrays</b>

Exemplo:

~~~
const notas = [10, 6, 8, 5.5, 10];

notas.pop(); // método pop remove o ultimo elemento da array

consta media = (notas[0] + notas[1] + notas[2] + notas[3]) / notas.length;

~~~

Exemplo:

~~~

<b>Desestruturar uma lista</b>

Exemplo:

~~~
const alunos = ['Joao', ´Juliana', 'Caio', 'Ana'];
const medias = [10, 8, 7, 9];
const lista = [alunos, medias];

function exibeNomeENota(aluno) {
	if(lista[0].includes(aluno)) {
		const [alunos, media] = lista; //desestrutura a lista e se torna uma variavel
		const indice = alunos.indexOf(aluno); //agora pode referencia alunos ao invés de lista
		const mediaAluno = medias[indice]; //agora pode referencia medias ao invés de lista
		console.log('${aluno} tem media ${mediaAluno}');
	} else {
		console.log('Estudante não existe na lista');
	}
}
~~~

<b>Percorrer uma lista - For clássico</b>

Exemplo 1:

~~~
const notas = [10, 8.5, 5, 6.5, 8, 7.5];

//primeira expressão: executada apenas uma vez
//segunda expressão: condição de execução
//terceira expressão: executada sempre ao final do bloco

for (let i = 0; i < notas.length; i++) {
	console.log(i, notas[i]);
}
~~~

Exemplo 2 (média com for): 

~~~
const notas = [10, 6.5, 8, 7.5];

let somaDasNotas = 0;

for (let i = 0; i < notas.length; i++) {
	somaDasNotas += notas[i];
}

const media = somaDasNotas / notas.length;
console.log('a média das notas é ${media}');
~~~

<b>Calcular Média - forEach</b>

Esse é um método de retorno que é usualmente chamado de callback, o metódo geralmente acompanhando de uma função dentro de outra função, só que a execução do método em si ele retorna undefined.

Exemplo : 

~~~
const notas = [10, 6.5, 8, 7.5];

let somaDasNotas = 0;

notas.forEach(function(nota) {
	somaDasNotas += nota;
})

const media = somaDasNotas / notas.length;
console.log('a média das notas é ${media}');
~~~

<b>Adicionar ponto extra - map</b>

O método map também tem a mesma funcionalidade de ser um método callback

Exemplo:

~~~

const notasAtualizadas = notas.map((nota) => return nota++ >= 10 ? 10 : nota++);

console.log(notasAtualizadas);
~~~

<b>Filtrando arrays = filter</b>

Esse também é um método call-back que filtra o array

Exemplo 1:

~~~
const alunos = ['Ana', 'Marcos', 'Maria', 'Mauro'];
const medias = [7, 4.5, 8, 7.5];

const tamanhoNome = alunos.filter((aluno) => {
	return aluno.length < 4;
})

console.log(tamanhoNome);
~~~

Exemplo 2:

~~~
const alunos = ['Ana', 'Marcos', 'Maria', 'Mauro'];
const medias = [7, 4.5, 8, 7.5];

const reprovados = alunos.filter((aluno, indice) => {
	return medias[indice] < 7;
})

console.log(reprovados);
~~~

<b>Somar com reduce</b>

Esse é um método callback, que reduz os elementos de um array para um elemento só

Exemplo 1:

~~~
const salaJS = [7, 8, 8, 7, 10, 6.5, 4, 10, 7];
const salaJava = [6, 5, 8, 9, 5, 6];
const salPython = [7, 3.5, 8, 9.5];

function calculaMedia(listaDeNotas) {
	const somaDasNotas = listaDeNotas.reduce((acumulador, nota) => {
		return acumulador + nota;
 	}, 0);

	const media = somaDasNotas / listaDeNotas.legth;
	return media;
}

console.log(calculaMeida(salaJS));
console.log(calculaMeida(salaJava));
console.log(calculaMeida(salaPython));
~~~

Exemplo 2:

~~~
const salaJS = [7, 8, 8, 7, 10, 6.5, 4, 10, 7];
const salaJava = [6, 5, 8, 9, 5, 6];

<b>Clonar Arrays - Spread Operator</b>

Exemplo:

~~~
const notas = [7, 7, 8, 9];
const novaListaNotas = [...notas. 10]; // spread operator - operador de espalhamento de valores, ao alterar essa variável ele não vai alterar na variável que está sendo referenciado

console.log(notas);
console.log(noaListaNotas);
~~~

<b>Removendo valores repetidos da lista - Set</b>

Set é um conjunto que armazena valores de dados único, é tipo um array, mas não é um array

Exemplo 1 :

~~~
consta nomes = ["Ana", "Clara", "Maria", "Maria", "Joao", "Joao", "Joao"];

const nomesAtualizados = new Set(nomes);

const listaNomesAtualizados = [...nomesAtualizados]; // transformar o set para array utilizando spread operator

console.log(nomesAtualizados);
console.log(listaNomesAtualizados);
~~~

Exemplo 2:

~~~
consta nomes = ["Ana", "Clara", "Maria", "Maria", "Joao", "Joao", "Joao"];

const listaNomesAtualizados = [...new Set(nomes)]; // transformar o set para array utilizando spread operator de forma refatorada

console.log(listaNomesAtualizados);
~~~
